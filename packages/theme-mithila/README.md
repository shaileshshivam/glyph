# @glyph/theme-mithila

The Mithila theme for Glyph. Based on Madhubani folk art traditions from Bihar, India: vermilion, saffron, Cormorant Garamond, and motifs for lotus, peacock, fish, surya, yantra, and kachni lattice.

## Install

```bash
pnpm add @glyph/theme-mithila
```

## Usage

```ts
import '@glyph/theme-mithila/theme.css';
```

Then use `@glyph/ui` components as normal — they read from the CSS variables this theme defines.

## Glyph components

```tsx
import { Lotus, Peacock, Fish, Surya, Yantra } from '@glyph/theme-mithila';
```
