# @glyph/ui

React component library for the Glyph CMS — Base UI primitives, CodeMirror editor, themeable via CSS variables.

## Install

```bash
pnpm add @glyph/ui react react-dom
```

Apply a theme stylesheet on the page:

```ts
import '@glyph/theme-mithila/theme.css';
```

## Components

Primitives: `Button`, `Input`, `Textarea`, `Label`, `Field`, `Select`, `Dialog`, `Toast`.
Editor: `CodeEditor` — CodeMirror 6 with rose-pine themes.

See `apps/ui-preview` in the monorepo for a running gallery.
